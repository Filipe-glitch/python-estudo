# 🐍 Estudos em Python - Lógica de Programação

Repositório dedicado aos meus estudos iniciais em **Python**.
## 🚀 Objetivo

Utilizar Python como linguagem principal e desenvolver a lógica de programação.

## 🛠️ Tecnologias utilizadas

- Python
- Pycharm e VS Code(IDEs)

## 📚 Conteúdos estudados

Durante este repositório, foram abordados:

- Estruturas condicionais (`if`)
- Estruturas de repetição (`for`, `while`)
- Controle de fluxo (`match-case` / similar ao switch)
- Funções
- Funções lambda
- Manipulação de strings
- Métodos de string
- Arrays e listas
- Dicionários
- Tratamento de exceções (`try/except`)

## 💡 Observações

- Este repositório é focado em lógica de programação
- Ainda não foram aplicados conceitos de Programação Orientada a Objetos (POO)

## 📌 Aprendizados

- Desenvolvimento do raciocínio lógico
- Resolução de problemas
- Estrutura básica de programas em Python
